Python Interpreter
==================

Install Python


Mac OS X
--------

Use Homebrew

.. code:: bash

    brew install python3
    brew install python


Linux (Debian/Ubuntu)
---------------------


.. code:: bash

    aptitude install python3.4
    aptitude install python2.7


Windows
-------

use Windows-Installer from:

* https://www.python.org/downloads/release/python-342/
* https://www.python.org/downloads/release/python-279/
